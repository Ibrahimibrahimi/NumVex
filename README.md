# NumVex

**NumVex** is a Java library that simulates the core functionality of [NumPy](https://numpy.org/) — the famous Python numerical computing library. It brings fast, expressive, and intuitive N-dimensional array operations to the Java ecosystem.

> Think of it as: `import numpy as np` → `import com.numvex.NumVex;`

## 🚀 Quick Start (Maven package)

### 1. Build the library

```bash
mvn package
```

### 2. Install to local Maven repo

```bash
mvn install
```

### 3. Add to your project's `pom.xml`

```xml
<dependency>
    <groupId>com.numvex</groupId>
    <artifactId>numvex</artifactId>
    <version>1.0.0</version>
</dependency>
```

### 4. Use it in your code

```java
import com.numvex.NumVex;
import com.numvex.NdArray;

public class Main {
    public static void main(String[] args) {

        // Create arrays
        NdArray a = NumVex.array(new double[]{1, 2, 3, 4, 5});
        NdArray b = NumVex.ones(5);
        NdArray z = NumVex.zeros(5);

        // Math operations
        NdArray result = NumVex.add(a, b);
        NdArray scaled = NumVex.multiply(a, 2.0);

        // comming soon
    }
}
```

## 🧠 Architecture

```
User Code
   │
   ▼
NumVex.java          ← the only class the user interacts with
   │
   ├──► NdArray.java        ← holds data, shape, dtype
   │
   └──► ops/
          └── MathOps.java    ← element-wise and math
                                 (more coming soon...)
```

`NumVex` acts as a **facade** — it never implements logic itself, it simply delegates to the right `ops` class. Users never need to import `ops` directly.
