# NumVex

**NumVex** is a Java library that simulates the core functionality of [NumPy](https://numpy.org/) — the famous Python numerical computing library. It brings fast, expressive, and intuitive N-dimensional array operations to the Java ecosystem.

> Think of it as: `import numpy as np` → `import com.numvex.NumVex;`

---

## ✨ Features
